# Instalacion 
```
npm i --save @nestjs/jwt passport-jwt bycrpt
npm i --save-dev @types/passoport-jwt
```

# Arrancamos proyecto
```
npm tun start:dev
```
# Modules & Resourse
```
nest g mo modules/auth
nest g res modules/users
```
# Controlers & Services
```
nest g co modules/auth
nest g s modules/auth 
```
